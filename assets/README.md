## Design Guidelines

- **Primary Color**: #00C853 (Emergency Green)
- **Secondary Color**: #FFFFFF (White)
- **Style**: Clean, modern, minimal
- **Theme**: Emergency, safety, trust
- **Icon Elements**: Consider using:
  - Number "1"
  - Phone symbol
  - Emergency shield
  - Life-saving symbol
